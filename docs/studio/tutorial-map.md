# Tutorial Map

## Dashboard

- Scope: Run health, spend, and workspace snapshots
- In-app location: `/`
- Outcome: Operator can identify active incidents and audit throughput

## Launch Scan

- Scope: URL, optional manual source, workspace naming, config selection, testing mode
- In-app location: `/scans/new`
- Outcome: Operator can start URL-first workflows safely

## Workspace Operations

- Scope: Status timeline, findings triage, workflow log interpretation
- In-app location: `/workspaces` and `/workspaces/[workspace]`
- Outcome: Operator can monitor and debug runs

## Report Handling

- Scope: Evidence reading, markdown copy/export, severity pass
- In-app location: `/workspaces/[workspace]`
- Outcome: Operator can convert findings to actionable tickets

## Full Tutorials Index

- Scope: End-to-end runbook and legal guardrails
- In-app location: `/tutorials`
- Outcome: New users can reach productive usage without external docs
